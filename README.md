# 💊 Medicine Management Application

## 📌 Overview

This project is a full-stack application to manage medicines with features like expiry tracking and stock monitoring.

---

## 🛠 Tech Stack

* Frontend: React + TypeScript + Ant Design
* Backend: .NET Web API (Clean Architecture)

---

## 🚀 Features

* View medicines in grid layout
* Search medicines
* Expiry alerts (Red highlight if < 30 days)
* Low stock alerts (Yellow highlight if quantity < 10)

---

## 🧱 Architecture

Backend follows Clean Architecture:

* Domain
* Application
* Infrastructure
* API

---

## ▶️ How to Run

### Backend

```bash
cd medicine-management-api
dotnet run
```

### Frontend

```bash
cd medicine-management-ui
npm install
npm start
```
